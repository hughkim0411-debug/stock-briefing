# 증시 브리핑 명령어

`stock-briefing` 스킬을 즉시 트리거합니다.

## 사용법

```
/증시브리핑
/증시브리핑 카카오톡
/증시브리핑 핵심만
```

- 인자 없음: 표준 심층 보고서 (.docx) 생성
- `카카오톡`: 보고서 생성 + 카카오톡 요약 자동 전송
- `핵심만`: 빠른 채팅 요약만 (.docx 생략)

## 실행 지시

이 명령어가 호출되면 즉시 다음을 수행하세요:

1. `stock-briefing` 스킬의 절차를 따릅니다.
2. **반드시 두 서브에이전트(hankyung-scraper, naver-news-searcher)를 한 메시지에서 병렬 호출**합니다.
3. 인자가 "카카오톡"이면 보고서 생성 후 자동으로 3개 메시지로 분할 전송합니다.
4. 인자가 "핵심만"이면 .docx 생성을 건너뛰고 채팅 요약만 반환합니다.

자세한 사항은 `stock-briefing` 스킬의 SKILL.md를 참조하세요.
