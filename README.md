# 📊 stock-briefing 플러그인

한국·미국 증시 일일 브리핑을 **병렬로 수집·교차 검증**한 뒤 **초보자도 이해할 수 있는 .docx 보고서**로 만들어주는 Claude Cowork 플러그인입니다.

---

## 핵심 특징

| 특징 | 설명 |
|---|---|
| 🚀 **병렬 수집** | 한국경제(웹 스크래핑)와 네이버 뉴스(검색 API)를 동시에 호출 → 시간 단축 |
| ✅ **교차 검증** | 두 소스의 핵심 수치(코스피, 다우, 환율, 금리)를 비교해 일치 여부 표시 |
| 📖 **초보자 친화** | 모든 전문 용어를 비유로 풀이, 코스피/국채금리/외국인이 뭔지부터 설명 |
| 📄 **.docx 자동 저장** | 사용자 워킹 폴더에 `증시브리핑_YYYYMMDD.docx` 형태로 저장 |
| 💬 **카카오톡 옵션** | 핵심 요약을 안전한 분량(메시지당 500자 이하)으로 분할 전송 |

---

## 디렉토리 구조

```
stock-briefing/
├── .claude-plugin/
│   └── plugin.json                  # 플러그인 매니페스트
├── agents/
│   ├── hankyung-scraper.md          # 서브에이전트 ① Claude for Chrome 기반
│   └── naver-news-searcher.md       # 서브에이전트 ② Naver Search MCP 기반
├── skills/stock-briefing/
│   ├── SKILL.md                     # 메인 스킬 (오케스트레이션)
│   └── scripts/
│       └── build_report.py          # .docx 생성기 (python-docx)
├── commands/
│   └── 증시브리핑.md                  # /증시브리핑 슬래시 명령어
└── README.md
```

## 작동 원리

```
사용자 요청 ("오늘 증시 어때?" 또는 /증시브리핑)
        ↓
   [메인 스킬 시작]
        ↓
   ┌────┴────┐  ← 한 메시지에서 두 Agent를 동시에 호출 (병렬!)
   ↓         ↓
[Agent A]  [Agent B]
한국경제    네이버 뉴스
스크래퍼    검색기
   ↓         ↓
   └────┬────┘
        ↓
   교차 검증 (수치 일치 여부 표 생성)
        ↓
   .docx 보고서 생성 (build_report.py)
        ↓
   채팅 요약 + 파일 링크 반환
        ↓
   (옵션) 카카오톡 3개로 분할 전송
```

## 설치 방법

### Cowork에서 사용 (로컬)

1. 이 폴더 전체를 다음 위치에 복사:
   - macOS: `~/Library/Application Support/Claude/local-agent-mode-plugins/`
   - Windows: `%APPDATA%\Claude\local-agent-mode-plugins\`
2. Cowork를 재시작
3. 플러그인 메뉴에서 `stock-briefing` 활성화

### Claude Code에서 사용

1. 이 폴더 전체를 다음 위치에 복사:
   - `~/.claude/plugins/stock-briefing/`
2. `claude plugin list`로 확인
3. 자동 로드됨

## 사용 예시

### 자연어 트리거

다음 문장 중 하나만 입력하면 스킬이 자동 호출됩니다:

- "오늘 증시 어때?"
- "코스피 코스닥 정리해줘"
- "주식 시장 브리핑 만들어줘"
- "한국 미국 증시 분석해줘"

### 슬래시 명령어

```
/증시브리핑              # 표준 보고서 생성
/증시브리핑 카카오톡     # + 카카오톡 자동 전송
/증시브리핑 핵심만       # 채팅 요약만 (빠른 모드)
```

## 의존성

이 플러그인은 다음 MCP·도구가 활성화되어 있어야 작동합니다:

| 필요 도구 | 용도 |
|---|---|
| **Claude for Chrome MCP** | 한국경제 사이트 스크래핑 (hankyung-scraper) |
| **Naver Search MCP** | 네이버 뉴스 검색 (naver-news-searcher) |
| **Python 3 + python-docx** | .docx 보고서 생성 (`pip install python-docx`) |
| **KakaotalkChat MCP** (선택) | 카카오톡 요약 전송 |

## 보고서 구성 (총 9개 장)

| 장 | 내용 |
|---|---|
| 0장 | 꼭 알아야 할 7가지 단어 (용어 풀이) |
| 1장 | 한 문단 요약 (색깔 박스) |
| 2장 | 한국 증시 — 숫자 + 외국인 매도 + 종목별 |
| 3장 | 미국 증시 — 3대 지수 + 매그니피센트 7 |
| 4장 | 두 가지 핵심 원인 (금리·반도체, 비유 포함) |
| 5장 | 오늘 새로 등장한 변수 |
| 6장 | 전문가 시각 (증권사 5곳 인용) |
| 7장 | **앞으로 어떻게 될까** — 시나리오 3개 + 악재 6개 + 호재 + 행동 지침 + 결론 |
| 부록 | 용어 사전 + 자료 검증 메모 |

## 출처 정책

- 모든 수치는 한국경제 또는 네이버 검색 결과 중 한 곳 이상에서 확인된 것만 사용
- 두 소스가 다르면 가장 최근 pubDate 채택, 차이는 보고서에 명시
- 인용된 기사는 채팅 응답 마지막에 `Sources:` 섹션으로 링크
